- [Weather Data Analysis](https://github.com/zabolot7/Weather-Data-Analysis): analysis and visualisation of weather data, written in Python using Pandas and Bokeh libraries
- [Linear Discriminant Analysis visualization](https://github.com/zabolot7/LDA-visualisation): an implementation and visualization of LDA, written in Python using Numpy and Matplotlib libraries
